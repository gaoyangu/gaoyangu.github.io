代码学习笔记：源码学习：amr-lmpcc

测试环境：

  • Ubuntu 16.04
  • ROS Kinetic

项目中的Running LMPCC 为仿真环境下的过程，本文为实际环境使用时遇到的问题总结。

1. Instalation instructions

Please follow the following instructions:

sudo apt install ros-kinetic-people-msgs
cd [workspace]/src
git clone https://github.com/bbrito/pedsim_ros.git -b four_persons
git clone https://github.com/spencer-project/spencer_messages.git
git clone https://github.com/srl-freiburg/spencer_tracking_rviz_plugin.git
git clone https://github.com/tud-amr/amr-lmpcc.git
cd ../
rosdep install --from-paths src --ignore-src -r -y
catkin_make
source devel/setup.bash

1.1 catkin_make 时遇到的问题

  1. 错误信息❌：fatal error: gazebo-9/gazebo/physics/physics.hh: No such file or directory
/catkin_ws/src/pedsim_ros/pedsim_gazebo_plugin/src/actor_poses_plugin.cpp:8:46: fatal error: gazebo-9/gazebo/physics/physics.hh: No such file or directory

解决方案：

📃pedsim_ros/pedsim_gazebo_plugin/src/actor_poses_plugin.cpp

#include <gazebo-9/gazebo/physics/physics.hh>
#include <gazebo-9/gazebo/util/system.hh>

将gazebo-9改为gazebo-7。

  1. 错误信息❌：fatal error: pedsim_msgs/TrackedPersons.h: No such file or directory
/catkin_ws/src/pedsim_ros/pedsim_gazebo_plugin/src/actor_poses_plugin.cpp:16:39: fatal error: pedsim_msgs/TrackedPersons.h: No such file or directory

解决方案：

  • 参考catkin_make

catkin_make -DCATKIN_WHITELIST_PACKAGES="pedsim_msgs"
  1. revert back to building all packages
