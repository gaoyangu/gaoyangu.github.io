2.2.2 ScansMerger::updateParam()

首先获取活动状态标志位，并加载参数

判断活动状态是否发生改变

如果状态为active，则订阅和发布话题

订阅的话题：

  • front_scan
  • rear_scan

发布的话题：

  • scan
  • pcl

否则，关闭所有的话题

2.2.3 ScansMerger::frontScanCallback()

首先进行 tf变换：laser ->fixed_frame_id，并将LaserScan转换为PointCloud

则根据标志位的信息将数据进行保存
