汇总：ROS学习资源

本文最后更新于：2020年4月1日 晚上

1. 重要网站

  1. ROS Wiki：官方网站，不多说
  2. 创客制造：几乎相当于中文版的ROS Wiki，但是内容要更加丰富一些
  3. ROS answers：问答网站，遇到的问题几乎都有，请用英文搜索
  4. 古月居：胡春旭的博客

2. 学习教程

2.1 入门视频教程

  1. 中科院软件所-机器人操作系统入门

    • 总时长：5个小时左右
    • 记得使用中国第一大视频学习网站：b站
    • 视频内容为介绍+演示，时间不长但是比较全面，学了一段时间再回过头来看，还是有很多收获
    • 课程讲义：Gitbook
    • 代码示例：Github

  2. 古月居：ROS入门21讲

    • 总时长：7个小时左右
    • 从安装Linux系统开始，手把手教学，涵盖了ROS Wiki核心教程入门阶段的所有内容
    • 课件&源码：Github

  3. 古月居：ROS理论与实践

    • 总时长：20个小时左右
    • 相比于入门教程，这个教程的内容更加的丰富和具体，涵盖的内容也是方方面面，比如机器人建模，视觉处理，SLAM建图等
    • 学习资源就需要大家发挥主观能动性了

2.2 文字版教程

  1. ROS培训教程(第2期)：onyrobotics(汤尼机器人)推出的ROS系列教程
  2. ROS学习相关电子书：古月的视频教程中也提到了这些书，实体书不好买，可以作为参考

3. 其他

3.1 博客

  1. 无处不在的小土：偶然发现的一个宝藏博客，博主写了很多ROS方面的系列教程，相对完整而且细致入微。
  2. 易科机器人实验室：智能移动机器人开发者精英自发组建的开放、开源、协作组织，但是许久不更新了

3.2 IDE

RoboWare Studio：基于VScode为ROS设计的，代码自动提示功能真的好用

RoboWare使用手册v0.7.1：软件上手其实不难，不过碰巧找到了手册

3.3 视频

  1. ROSCon 2019 机器人操作系统开发者大会视频集
  2. The Construct：西班牙的一家做ROS培训的机构
